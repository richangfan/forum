# 数据格式
HTTP请求和响应都使用"application/json"格式

# 接口列表
请求成功返回的http body
```
{
    error: 0,
    message: string,
    data: object
}
```
请求失败返回的http body
```
{
    error: 1,
    message: string,
    data: object
}
```
